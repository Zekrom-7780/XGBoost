Provide the parameters for XGBoost

Default Parameters list

(All the parameters provided are for [booster=gbtree] method.

Default Parameters list

  • booster [default= gbtree]
  • verbosity [default=1]
  • validate_parameters [default to false, except for Python, R and CLI interface]
  • nthread [default to maximum number of threads available if not set]
  • eta [default=0.3, alias: learning_rate] (eta range=[0,1])
  • gamma [default=0, alias: min_split_loss] (gamma range=[0,infinity])
  • max_depth [default=6] (range: [0,∞])
  • min_child_weight [default=1] (range=[0,infinity])
  • max_delta_step [default=0]
  • subsample [default=1] (range=[0,1])
  • sampling_method [default= uniform]
  • colsample_bytree, colsample_bylevel, colsample_bynode [default=1]
  • lambda [default=1, alias: reg_lambda]
  • alpha [default=0, alias: reg_alpha]
  • updater [default= grow_colmaker,prune]
  • refresh_leaf [default=1]
  • process_type [default= default]
  • grow_policy [default= depthwise]
  • max_delta_step [default=0]
  • max_leaves [default=0]
  • max_bin, [default=256]
  • predictor, [default=``auto``]
